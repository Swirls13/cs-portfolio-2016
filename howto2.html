INPUT

In the "body" tags you have to add two “input” boxes. These boxes are where you can type in the numbers for your calculator.

In the input tag, there are two things to take note of. The “type” and the “id”. These two features are very important when it comes to making all of this.

  • The type is used to make the user limited to only inputting numbers into the boxes.
  • The id is used to identify the element in places like CSS or JavaScript and manipulate it into changing its look or doing specific things
  • The id should be completely unique to its name in order to tell the difference between it and other objects
