How To

What is HTML?

HTML is an abbreviation for Hyper Text Markup Language and it's the standard markup language for creating Web pages by using markups to describe the structures of the web pages. Elements within the HTML pages are represented by tags, which help label pieces of content such as "paragraphs", "title", "tables" and much more. The tags aren't shown within the actual website, but are rather used to render the page contents.

To start off your website, you need the basic format for it. Below is the basic formatting for coding:

  • To start out, you should always have the tags < HTML > and < /HTML >
  • Next is the < body > < /body >. This tag is important because it helps indicate where to keep the contents of the page in.
  • The < h1 > < /h1 > tag indicates the heading of the website. There are numbers h1 to h6 to change the size of the heading from most important (h1) to least important (h6).
  • < p > < /p > indicates the paragraphs, where most of the writing is put in. Whatever you type in the code will show up on your website.

There are a lot of different ways to approach making your website. Since we now know the basics of it, we can now learn how to add different contents into the webpage to make it more interesting.

Images

Images are one of the cool things you can put on your website. It helps liven up the page, making it more attractive to look at.

  • To put an image you start off with the tag "< img src="" >"
  • The easiest way to insert a photo is by going on google images and finding a photo you would like to add to your website. After finding a photo or gif you like, you copy it's image address and paste it into the quotations of the tag.
  • ie: < img src "http://orig08.deviantart.net/6fa9/f/2015/065/2/7/rainbow_unicorn___charity_auction_by_dragonsandbeasties-d8knxwn.jpg" > >
  • After inserting the photo, you might notice some things about it you don't like. The photo may be too big or small, but there's a way to fix that
  • To do this, you'll need to add "width" (width="") and/or "height" (height) in your tag. You can put it anywhere within the tag, as long as it doesn't interfere with the code. You can make the photo however big to your liking.
  • ie: < img src="http://orig08.deviantart.net/6fa9/f/2015/065/2/7/rainbow_unicorn___charity_auction_by_dragonsandbeasties-d8knxwn.jpg" width="400" height="350" >
