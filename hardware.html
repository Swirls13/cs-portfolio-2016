Hardware

Motherboard

The Motherboard is the "heart of the computer”, or the computer’s central, which is usually found within PCs. It lets the different parts of the computer to communicate with each other. Before, in the 1980’s motherboards were quite large in size, but as technology advanced sizes of the motherboard decreased, making it the size it is today.

CPU

The Central Processing Unit, aka the CPU, is known as the "brains” of the computer. It’s a very important part of the computer because it’s where most calculations take place. The CPU usually needs one or more printed circuit for larger machines, but when it comes to personal computers, it’s usually a single chip called a microprocessor. CPUs today are small, square and have many metallic connectors or pins on it’s underside. There are two typical components to it, the Arithmetic Logic Unit, which performs arithmetic and logical operations, and the Control Unit, which extracts instructions from memory and decodes and executes them.

RAM

RAM, or Random Access Memory, is a type of computer memory that can be randomly accessed, being a common type of memory found in computers and other devices. It's a type of hardware device that allows information to retrieved and stored onto the computer. There are two types of RAMs, the DRAM (Dynamic Random Access Memory) and SRAM (Static Random Access Memory). DRAM is most commonly used, but it needs to be refreshed a lot. The SRAM, however, is not as common due to it being more expensive. Though, it's much faster, so you don’t need to refresh.

Hard Drive

Hard Drive is actually an abbreviation for "Hard Disk Drive”, it is a device that is used to permanently store and receive information. Usually, hard drives consists of one or more platters which data is written. Data is able to be sent to and from the drive, which helps tell the hard drive what to do and how to move the components in the drive. When the operating system needs to read or write information, it will need the hard drive to determine the file location and available write areas.

Monitor, Keyboard, and Mouse

A monitor is a piece of the computer hardware that displays video and graphics informations which is generated by the computer.

A keyboard is used to let the operator insert a command into the PC.

A mouse is an input device that sends information to the computer with just a click of a button. It's an alternative way to interact with the computer besides the keyboard.
